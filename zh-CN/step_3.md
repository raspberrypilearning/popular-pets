## 挑战：创建你自己的条形图

你可以按类似方式创建条形图。只需使用 `barchart = pygal.Bar()` 来创建一个新的条形图，然后添加数据并按与饼图同样的方式呈现出来。

向你的 Code Club 成员收集数据来创建你自己的条形图。

请务必选择一个大家都了解的话题！

以下是一些想法：

+ 你最爱的运动是什么？
+ 你最爱的冰淇淋口味是什么？
+ 你怎么去学校？
+ 你的生日在哪个月份？
+ 你玩我的世界吗？（是／否）

请不要问一些需提供个人资料的问题，如大家住在哪里。如果你不确定，请询问你的俱乐部领导。
